# 설명
> 1주일간 동료 팀원과 협업해서 만든 어플입니다. 학원에 들어올 때 쓰는 각오문의  기수와 이름(텍스트)을 firebase-ml-vision을 통해 인식합니다. 인식한 이름으로 크롤링해서 해당 팀원의 최신 작품 영상을 가져옵니다. 그리고 보여지는 바닥을 클릭하면 가져온 영상을  클릭한 장소에 AR로 보여줍니다
